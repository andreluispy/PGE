# PGE
 PGE(Python Game Engine), an easy alternative to pygame

 Versions:
 
 - 2D Engine(SDL using PyGame)
 - 3D Engine(Coming Soon)

# Starting
 - Download engine dist in: [Click Here]()
 - Extract engine folder in your folder project
 - Dev with Docs!
